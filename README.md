# Andy-Desktop-Assistant
Andy Desktop Assistant is a Voice-assistant Program made using Python where you can talk to Andy and ask anything. You can ask Andy to open youtube and search anything or open any website or ask a fact. This project can also do many tasks such as taking screenshot, recording video, shutting down computer, renaming or making new file or folder. It can also summarize any pdf file or you can even ask it to compress an image by saying andy to compress an image named anything. You can also ask it to make color of whole computer screen change like disco by Andy just by speaking.
